二元体上の行列演算

二元体 \(\mathbb{F}_2\)

\(\mathbb{F}_2=\{0,1\}\)

\(\mathbb{F}_2\)の演算は、整数として計算した後、 計算結果が偶数なら\(0\)、奇数なら \(1\)と置き換えればよい。

特に、任意の\(x \in \mathbb{F}_2\)に対し\(x+x=0\)が成立する。

\(\mathbb{F}_2\)上の行列の演算

通常の行列の演算と同じ。但し加法、乗法の計算規則は、\(\mathbb{F}_2\)に従う。 \[ \begin{pmatrix} 0&1&1\\ \end{pmatrix} \begin{pmatrix} 1&1&0\\ 1&0&0\\ 1&1&1 \end{pmatrix} =\begin{pmatrix} 0&1&1 \end{pmatrix} \]

数式入力の例

\[ \tanh x = \frac{ e^{x} - e^{-x} } { e^{x} + e^{-x} } \] \[\left. \frac{1}{s^2 - s + 1} \right|_{s = 2} = \frac{1}{3}\] \[\sqrt{ a + b + 2\sqrt{ab} } = \sqrt{a + b} \] \[\int_0^1 x^2 dx= \left[ \frac{x^3}{3} \right]_0^1=\frac{1}{3} \]
